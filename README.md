# WiseSaying (명언)
#### 멋쟁이사자처럼 토이프로젝트

<br>

## 과제
#### 1. console에서 작동되는 명언 프로그램을 만들어라
#### 2. 단계별로 순차적인 개발을 해라

<br>

## 요구사항
### 1단계
#### 프로그램 종료
```
== 명언 앱 ==
명령) 종료
```

<br>

### 2단계
#### 데이터 등록
```
== 명언 앱 ==
명령) 등록
명언 : 현재를 사랑하라.
작가 : 작자미상
명령) 종료
```

<br>

### 3단계
#### 등록 시 생성된 명언 번호 출력
```
== 명언 앱 ==
명령) 등록
명언 : 현재를 사랑하라.
작가 : 작자미상
1번 명언이 등록되었습니다.
명령) 종료
```

<br>

### 4단계
#### 등록할때 마다 생성되는 명언번호가 증가
```
== 명언 앱 ==

1번 명언이 등록되었습니다.
명령) 등록

작가 : 작자미상
2번 명언이 등록되었습니다.
명령) 종료
```

<br>

### 5단계
#### 목록 보기
```
== 명언 앱 ==
명령) 등록

작가 : 작자미상
1번 명언이 등록되었습니다.

작가 : 작자미상
2번 명언이 등록되었습니다.
명령) 목록
번호 / 작가 / 명언
----------------------
2 / 작자미상 / 과거에 집착하지 마라.
1 / 작자미상 / 현재를 사랑하라.
명령) 종료
```

<br>
